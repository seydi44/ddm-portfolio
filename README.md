# ddm-portfolio

Didem Gül için tek sayfa (static) portfolyo sitesi.

## Çalıştırma (lokal)
- VS Code'da klasörü aç
- Live Server eklentisi ile `index.html` dosyasını çalıştır

## İçerik güncelleme
- Metinler: `index.html`
- Stil: `styles.css`
- Davranışlar: `script.js`
- Görseller: `assets/` klasörü

> `assets/CV-Didem-Gul.pdf` dosyasını kendi CV'inle değiştir.
